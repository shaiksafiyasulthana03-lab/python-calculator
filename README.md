#Modular Mathematical Engine & CLI suite 

A robust command-Line interface (CLI)

##core Engineering Features
- **Algorithmic Exception Handling**: Built-in defensive code layers utilizing `try-except` blocks to intercept mathematical boundaries like division-by-zero errors and invalid input strings.
- **State Preservation & History Tracking**: Leverages dynamic linear data structures (`List`) to preserve a chronological log of valid mathematical calculations during a live user session.
- **Modular Object-Oriented Principles**: Clean separation of mathematical computational layers from the input-handling terminal interface to ensure codebase scalability.

## Technical Stack

- **Language**: Python 3.x
- **Concepts**: Functional Programming, Data Structures, Structural Input Validation, Control Flow Matrix

## 🛠️ How to Run Locally

1. Ensure you have **Python 3** installed on your system.
2. Clone or download this repository file.
3. Open your terminal or VS Code console inside the file directory.
4. Execute the application runner command:
   ```bash
   python calculator.py
   ```
